🚀 FPGA Workshop using PYNQ-Z2

📌 Introduction

This repository documents hands-on work completed during an FPGA workshop using the PYNQ-Z2 development board. The goal is to build a strong foundation in digital logic design through practical experiments.

Each experiment uses physical inputs (switches/buttons) and visual outputs (LEDs), making learning interactive.

🎯 Objectives

  • Understand fundamental digital logic concepts
  • Implement combinational circuits
  • Learn hardware-software interaction using Python (PYNQ)
  • Gain practical FPGA experience

🧩 Concepts Covered

🔹 Digital Logic Basics

  • AND, OR, NOT, NAND, NOR, XOR gates
  • Truth tables and logic expressions

🔹 Code Conversion

  • Binary to Gray code
  • Gray to Binary conversion

🔹 Arithmetic Circuits

  • Half Adder
  • Half Subtractor

🔹 Data Selection Circuits

  • Multiplexer (MUX)

🛠️ Hardware & Tools

🔧 Hardware

  • PYNQ-Z2 FPGA Development Board
  • USB cable

💻 Software

  • Jupyter Notebook (PYNQ environment)
  • Python

🔍 About the PYNQ-Z2

The PYNQ-Z2 is based on the Xilinx Zynq-7000 SoC, combining:

  • Processing System (PS): ARM Cortex-A9
  • Programmable Logic (PL): FPGA fabric

⚙️ Key Features

  • Xilinx XC7Z020 FPGA
  • 512 MB DDR3 RAM
  • MicroSD support
  • USB & Ethernet
  • HDMI I/O
  • Push buttons, switches, LEDs, RGB LEDs

🧪 How Experiments Work

  1. Read input from switches/buttons
  2. Process logic
  3. Display output using LEDs

Most programs run continuously:

while True:
    # read inputs
    # apply logic
    # update LEDs

▶️ Getting Started

  1. Connect PYNQ-Z2 to your laptop
  2. Power using USB
  3. Open Jupyter environment
  4. Run the code
  5. Use switches to test

⚠️ Notes

  • Programs may run indefinitely (while True)
  • Press Ctrl + C to stop
  • Ensure proper power connection
